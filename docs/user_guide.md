# User Guide
## Overview
Instructions for using the Amanuensis transcription app.

## Features
- **Split-Window UI**: View images and edit transcriptions side-by-side.
- **Projects**: Organize files into projects/collections.
- **Metadata**: Add/edit metadata (e.g., title, tags) via forms.
- **Templates**: Apply formatting (e.g., invoice, notes) to transcriptions.
- **Plugins**: Extend functionality (e.g., image filters, export tools).
- **Editor Layouts**: Support for tables, headers, and columnar layouts.
