# Template System Guide
## Managing Templates
- Create/edit templates via Project > Edit Templates or ProjectPanel context menu.
- Apply templates to files via ProjectPanel.
- Import external templates from src/templates/ (JSON).

## Built-in Templates
- Invoice: Fields for Invoice Number, Date, Amount.
- Meeting Notes: Fields for Date, Attendees, Notes.

## External Templates
- Format: {"name": "Template Name", "fields": [{"name": "Field Name", "type": "text/date/number/multiline"}]}
