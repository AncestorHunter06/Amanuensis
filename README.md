# Amanuensis Transcription App

A Tcl/Tk-based desktop application for transcribing scanned documents (e.g., historical records).

## Directory Structure
- **src/**: Tcl/Tk scripts for the app.
- **assets**: Sample images/PDFs for testing.
- **templates/**: Boilerplate layouts (e.g., census forms).
- **metadata/**: Document metadata (e.g., JSON).
- **docs/**: Documentation and notes.
- **tests/**: Test scripts and data.
